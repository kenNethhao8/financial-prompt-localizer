# Prompt Template Library

Use these templates when a Chinese request clearly falls into a known financial engineering workflow.

The templates are not meant to replace `SKILL.md`. They provide reusable prompt frames for common task families:

- `backtest.md`
- `factor_research.md`
- `portfolio_optimization.md`
- `financial_ml.md`
- `data_engineering.md`
- `financial_nlp.md`
- `derivatives_fixed_income_credit.md`

When localizing a Chinese request, choose the closest template, preserve the user's original constraints, and fill only the assumptions that are safe to infer.
